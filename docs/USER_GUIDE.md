# Руководство пользователя - Map Virtual Tours

## Добро пожаловать!

**Map** - это интерактивная система виртуальных туров по зданиям Курского государственного университета. Здесь вы можете ознакомиться с факультетом физики, математики и информатики через панорамные изображения и интерактивные планы.

## Навигация по сайту

### Главная страница
- **URL**: `/`
- **Описание**: Приветственная страница с описанием доступных виртуальных туров
- **Возможности**:
  - Обзор туров по этажам
  - Переход к детальным планам

### Планы эвакуации
- **URL**: `/evac_plans/`
- **Описание**: Галерея всех доступных планов эвакуации
- **Возможности**:
  - Просмотр миниатюр планов
  - Переход в режим редактирования (для администраторов)

### Админ-панель
- **URL**: `/admin/`
- **Описание**: Инструмент для создания и редактирования виртуальных туров
- **Доступ**: Только для авторизованных пользователей

## Создание виртуального тура

### Шаг 1: Создание плана
1. Перейдите в админ-панель (`/admin/`)
2. В левом меню "Создать план":
   - **Название**: введите понятное название (например, "1 этаж, главный корпус")
   - **Файл плана**: загрузите изображение плана здания (формат: JPG или PNG)
3. Нажмите **"Создать план"**

### Шаг 2: Добавление точек
1. Выберите шаг **"2. Точки"** в верхнем меню
2. В панели инструментов выберите **"Добавить точку"**
3. Кликните на план в месте расположения точки
4. В правой панели (инспектор):
   - Введите название точки
   - Нажмите **"Сохранить точку"**

### Шаг 3: Загрузка панорам
1. Выберите шаг **"3. Панорамы"**
2. Выберите точку в списке или на плане
3. В инспекторе:
   - Выберите файл панорамы (360° фото помещения)
   - Нажмите **"Загрузить / заменить"**

### Шаг 4: Создание переходов
1. Выберите шаг **"4. Переходы"**
2. Выберите точку с панорамой
3. В панели инструментов выберите **"Добавить переход"**
4. В инспекторе выберите целевую точку из списка
5. Кликните на панораму в месте, где должен быть маркер перехода

### Шаг 5: Проверка тура
1. Выберите шаг **"5. Проверка"**
2. Просмотрите список найденных проблем
3. Исправьте выявленные недочеты:
   - Добавьте недостающие панорамы
   - Создайте пропущенные переходы

## Инструменты редактирования

### Режимы работы
- **Просмотр**: навигация по плану и панорамам
- **Добавить точку**: размещение новых точек на плане
- **Добавить переход**: создание маркеров на панорамах

### Панели интерфейса

#### Левая панель (Сайдбар)
- **Список планов**: выбор активного плана для редактирования
- **Создание плана**: форма для добавления нового плана

#### Центральная область (Workspace)
- **План здания**: интерактивная карта с точками
- **Панорама**: 360° изображение выбранной точки
- **Панель инструментов**: переключение режимов работы

#### Правая панель (Инспектор)
- **Свойства плана**: редактирование названия
- **Свойства точки**: название и координаты
- **Управление панорамой**: загрузка/удаление изображений
- **Переходы**: список маркеров и выбор целей

## Просмотр готовых туров

### Базовый просмотр
1. Перейдите на главную страницу
2. Выберите интересующий тур
3. Кликайте на точки плана для перехода между панорамами
4. Используйте маркеры на панорамах для навигации

### Интерактивные элементы
- **Точки на плане**: клик для перехода к панораме
- **Маркеры на панораме**: клик для перехода к другой точке
- **Индикаторы**: точки с панорамой отмечены кругом, без панорамы - точкой

## Типичные проблемы и решения

### План не загружается
**Проблема**: Изображение плана не отображается
**Решение**:
- Проверьте формат файла (должен быть JPG или PNG)
- Убедитесь, что файл не поврежден
- Проверьте размер файла (не более 10 МБ)

### Точка не сохраняется
**Проблема**: После клика точка не появляется
**Решение**:
- Убедитесь, что план выбран
- Проверьте, что введено название точки
- Попробуйте перезагрузить страницу

### Панорама не загружается
**Проблема**: Панорамное изображение не загружается
**Решение**:
- Проверьте формат файла (JPG или PNG)
- Убедитесь, что файл является панорамой (360°)
- Проверьте размер файла

### Маркеры не отображаются
**Проблема**: Переходы не видны на панораме
**Решение**:
- Убедитесь, что выбрана точка с панорамой
- Проверьте, что созданы маркеры переходов
- Обновите страницу

## Советы по созданию качественных туров

### Подготовка материалов
1. **Планы**: Используйте четкие, контрастные изображения планов
2. **Панорамы**: Делайте сферические фотографии высокого качества
3. **Масштабирование**: Убедитесь, что планы имеют правильный масштаб

### Структура тура
1. **Логичность**: Создавайте естественные маршруты перемещения
2. **Полнота**: Каждая точка должна иметь панораму
3. **Связность**: Все точки должны быть связаны переходами

### Оптимизация производительности
1. **Размер файлов**: Сжимайте изображения без потери качества
2. **Формат**: Используйте JPG для фотографий, PNG для планов
3. **Количество**: Не перегружайте тур слишком большим количеством точек
