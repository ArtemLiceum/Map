Редактор плана эвакуации — добавление точек и панорам

1) Загрузить план (карта здания)

Можно сохранить на сервер через API (JPG или PNG).

2) План — добавить метку

Клик по картинке добавит метку. Метки сохраняются в процентах от размеров изображения.

Список точек

Инструкция

  1. Загрузите план.
  2. Кликните по месту — задайте имя точки и (опционально) загрузите панораму.
  3. Нажмите на маркер — откроется панорама; в ней можно добавить маркеры для перехода к другим точкам.
